# OpenAPI Studio Hover Utility Menus Design

**Date:** 2026-07-23

## Goal

OpenAPI Studio 상단 보조 작업의 조작 단계를 줄인다. 데스크톱에서는 메뉴 트리거에 포인터를 올리거나 키보드로 포커스하면 내용을 열고, 내보내기와 샘플의 항목 클릭은 즉시 다운로드를 실행한다.

## Scope

- `형식`, `내보내기`, `샘플` 트리거는 데스크톱의 호버와 키보드 포커스에서 팝오버를 연다.
- 터치 환경에서는 트리거 탭으로 메뉴를 열고, 내부 항목 탭으로 작업을 실행한다.
- 메뉴 밖 클릭, 포커스 이탈, Escape는 열린 메뉴를 닫는다.
- 한 번에 하나의 메뉴만 열린다.
- 샘플 메뉴는 선택 상자와 별도 다운로드 버튼을 제거하고, 버전별 다운로드 항목을 바로 표시한다.
- 형식 변환과 형식 강제 지정의 의미를 구분한다.

## Menu Content

### 내보내기

메뉴 항목은 `YAML 다운로드`, `JSON 다운로드`다. 각 항목은 현재 문서를 해당 형식으로 직렬화하고 즉시 다운로드한다.

### 샘플

메뉴에는 중복되는 `샘플` 라벨이나 선택 상자를 두지 않는다. 다음 네 항목을 항상 표시하며, 각각 클릭 즉시 해당 버전의 YAML 샘플을 다운로드한다.

| 표시 이름 | 내려받는 파일 |
| --- | --- |
| Swagger 2.0 | `swagger-2.0-sample.yaml` |
| OpenAPI 3.0.4 | `openapi-3.0-sample.yaml` |
| OpenAPI 3.1.2 | `openapi-3.1-sample.yaml` |
| OpenAPI 3.2.0 | `openapi-3.2-sample.yaml` |

### 형식

`YAML로 변환`, `JSON으로 변환`은 유효한 문서를 대상 형식으로 직렬화해 편집기 내용을 실제로 바꾼다. 현재 형식과 같은 대상은 비활성화한다.

자동 감지가 올바르지 않을 때 쓰는 기존 기능은 문서를 변환하지 않고 파서의 입력 형식만 지정한다. 따라서 `YAML로 강제 해석`, `JSON으로 강제 해석`은 각각 `YAML로 읽기`, `JSON으로 읽기`로 표기한다. `형식 다시 감지`도 유지한다.

## Interaction and Accessibility

`UtilityMenu`는 트리거 버튼과 메뉴 패널을 하나의 컨테이너로 감싼다. 트리거는 `aria-expanded`, `aria-haspopup="menu"`를 제공하고, 메뉴 패널은 `role="menu"`, 내부 실행 항목은 버튼으로 유지한다.

호버가 가능한 환경에서는 컨테이너에 포인터가 들어올 때 메뉴를 열고 벗어날 때 닫는다. 키보드 포커스가 컨테이너 안에 있는 동안 메뉴는 유지된다. 호버가 불가능한 환경에서는 트리거 탭으로 메뉴를 열며, 메뉴 밖 탭·Escape·작업 실행으로 닫는다. 다운로드·변환 작업을 실행한 뒤에는 메뉴를 닫는다.

## State and Interfaces

`Topbar`는 열린 메뉴 식별자만 로컬 상태로 관리한다. 샘플 선택 상태와 `onSampleVersion`은 제거하고, 샘플 다운로드는 각 항목에서 `onDownloadSample(version)`을 직접 호출한다.

`useWorkspace.convertFormat`은 목표 형식을 인자로 받도록 변경한다. `Topbar`는 `onConvertFormat(format)`으로 각 변환 항목을 직접 연결한다. `forceFormat`의 동작은 변경하지 않는다.

## Verification

- 컴포넌트 테스트는 샘플 선택 상자와 별도 샘플 다운로드 버튼이 사라지고, 네 버전별 다운로드 항목이 존재하는지 검증한다.
- E2E는 메뉴 트리거 호버로 항목이 표시되는지, YAML/JSON 내보내기 항목 클릭이 바로 다운로드되는지, 각 샘플 버전 항목이 올바른 YAML 파일을 내려받는지 검증한다.
- E2E는 형식 변환과 `읽기` 동작의 이름 및 목표 형식 변환을 검증한다.
- `mise run check`와 `git diff --check`가 모두 성공해야 한다.

## Non-goals

- 샘플 형식의 JSON 제공
- 메가 메뉴나 별도 다운로드 모달
- 문서 내용이나 변환 규칙 변경
